### 박재성 - Kakao-Pay-Book-Search

### 소개 
> 앱 이름 :  BookSearch   
> 앱 설명 : 카카오 도서 검색 API를 이용하여 도서 검색을 하는 어플리케이션   

### 버전
> minSdk : 23   
> targetSdk : 30   

### 개발환경
> 사용 언어 : Kotlin   
> 개발 IDE : Android Studio
